## 議程時段彈性

| 情境 | 處理 |
|------|------|
| 玉山方對「Why 3CX」段有大量追問（25 min 不夠） | 壓縮「商業條款」段為 2 min（留 NDA 後深談），多給 Q&A 8 min |
| 玉山方提早抓住核心、無需走完全部 25 頁 | 主講者可跳過 P15-16（Premium Partner / Q&A 兩頁），多給開放討論 15 min |
| 玉山方主管中途離席接電話 | 主講者繼續對在場其他人；主管回來後快速 5 句話 catch up |
| 玉山方意外要求現場 demo | 拒絕當場 demo（無實機）；改提議：5/15 安排專場 demo @ 瑛聲 demo lab |
